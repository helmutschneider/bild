# BILD
This is a BMP-decoder written in C. At some point I might write a PNG implementation too.

An example image viewer built with SDL3 is included.

## Usage
Compile the thing:
```shell
make
```

Open a BMP:

```
bin/bild [path_to_bmp_here]
```
